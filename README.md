# Github Viewer

![Github](github.png)

Github의 인기 저장소 목록을 확인할 수 있고, Github 사용자 간의 프로필을 대결 구도로 비교해 볼 수 있는 웹 어플리케이션입니다.

## Installation

```sh
npm install
```

## Development

```sh
npm start
# Visit localhost:3000
```

## Warning

🚨 Github API 요청이 많아지면 제한될 수 있습니다. 그럴 경우, `/utils/api`를 참고하여 Mock Data로 대체하여 사용하세요.
